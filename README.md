The Reaction Speed Tester is a simple JavaScript project that measures how quickly a user responds when the screen changes color. It uses randomized delays, DOM event listeners, and timestamp calculations to create a clean, fast reaction time game that works on both desktop and mobile. The interface stays minimal so the user can focus entirely on reacting, and the project is great for learning basic web interactivity, timing functions, and UI feedback. To use test your reaction time, you must click the "start test" button and wait for the big sqaure to turn green to click it.
